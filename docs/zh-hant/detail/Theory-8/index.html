不以物質獎勵來引導讀經

2015年夏天我和孩子之間有一個關於“冰激凌和跑步”的對話。暑假，我想開始帶孩子每天跑步。我在地圖上標出我們跑步的路線，孩子一看，哇，2.5英里（相當於4公里），再看看外面的太陽，孩子們都不情願了。

我就問他們：“如果給你們兩個選擇，冰激凌和跑步，你會選擇什麼？”孩子們不加思索地説：“冰激凌!”

接着我又問：“如果每天給你一個冰激凌，或者每天跑一次步，你會選擇什麼？”孩子們還是不加思索地説：“冰激凌!”

我又繼續問：“如果你每天一個冰激凌，堅持一年，兩年，三年下來，你有什麼收穫？”孩子們想了想説：“也許會變成一個大胖子。”我追着問：“除了變成一個大胖子，還有什麼收穫？”孩子們想了良久，搖搖頭。

然後，我換了另個選項問他們：“如果你每天跑一次步，堅持一年，兩年，三年下來，你有什麼收穫？”孩子們很快就回答：“這樣堅持跑步，身體會健康。”我又啓發他們説：“除了身體健康，你還會有什麼收穫呢？”孩子們也嘗試着回答：“也許會養成鍛鍊的習慣。”我説：“對的，經過這樣堅持下來，你們就體會到什麼是做事情的恆心和毅力。有了恆心和毅力，你可以把這些品質用到其他方面，讓你一輩子受益。”

經過我一番述説，孩子們同意去跑步了。但是他們馬上提出了一個要求：“能不能既要冰激凌，又要跑步。也就是每天跑完步，你獎勵我們一個冰激凌。”

我於是説：“這樣可能會帶來兩個問題。第一，你們在跑步的時候，老想着冰激凌，就不會體會跑步的快樂。媽媽是從小學一年級開始堅持每天跑步，到中學，再到大學，即使工作了也堅持跑步，我是真心喜歡跑步，我也希望你們能體會跑步的快樂。第二，如果你跑步是為了一個冰激凌，如果有那麼一天，我忘記了給你們買冰激凌，你們就沒有動力去跑步了，也就無法堅持下來了。”聽了這個分析，孩子們最後同意，不要冰激凌，就直接去跑步！

第一天跑下來，孩子們挺累的，我也感覺累；第二天，感覺稍微好點；第三天，慢慢適應了，就感覺沒有那麼累。就這樣，一天一天跑下來，慢慢地，孩子們也適應了。有時候，我下班回到家，孩子們就主動説：“媽媽，我們去跑步！”我有些累，就皺皺眉頭説：“媽媽有點累，不知道是否跑得動。”沒想到孩子們説：“不要緊，你可以騎車，我們跑步。”你看，他們自動提出要跑步了。

又有一天，孩子突然跟我説：“媽媽，我覺得跑步挺好的。”我問：“為什麼呢？”女兒説：“我們跑步的時候，看着藍天白雲，再看看路旁的花花草草，很開心！”兒子説：“我感覺跑步的時候，腳下生風，就像那個神行百步，草上飛！”

可以看到，孩子們開始體會到了跑步的樂趣。

其實讀經也一樣。我講這個例子，主要是想説明不要通過物質刺激來引導孩子讀經，而是家長要領着孩子進入狀態，讓讀經成為一種自然的狀態。當然，在讀經後，也要讓孩子們放鬆玩一下。

同於自性，頤養天真。有教無類，樂學致遠。
