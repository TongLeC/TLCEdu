學習中文，眼光有多遠，就能走多遠

中文學習目標的制定與評估很重要，因為這決定了人們為了實現目標而努力採取的行動。這個跟企業裏的目標管理，績效管理和戰略管理是異曲同工的。

説起目標與評估，對於一棵植物，人們最想看到的是花果；對於孩子的中文學習，家長最看重的是孩子認了多少字，能否自由閲讀，能否寫文章。這幾乎是一般家長問得最多的問題。要回答這個問題，我們首先思考兩個方面。

一、眼光看多遠

我們讓孩子學習中文，着眼點是在一個月呢，一年呢，還是五年、十年？相信很多家長會説是五年或十年後的效果。但現實是，人們更着重看一個月，或一年的。例如我剛開始教課，就有家長問我一年能教多少個漢字。我的回答是，如果家長的目光是盯着一年看的，那麼快速識字法最能告訴你一年記多少個字，但是卻不能保證孩子能否運用這些字，也不能保證兩三年後還是否記得這些字。這樣的效果就好比上面提到的那棵一開始就拼命結果子的樹，根沒有紮好，終不能長久。

現在的中文教學，基本是不重視“聽説”訓練的，而是直接教孩子認字，希望儘快進入閲讀和寫作。其結果則是欲速而不達。絕大部分海外中文學生的最終閲讀、寫作能力只達到初級水平，聽、説也沒練好。以至於很多中文班宣傳教學成果時，只能説三個月記了多少字，一年記了多少字。殊不知，沒法學以致用的記字，記憶能保持多久呢?我們也看到，有的孩子強記了不少漢字後，文化學習和聽説訓練跟不上，最終的結果是不但沒法達到自由閲讀，更有甚者是厭惡中文，再也不願拿起中文書。

站在理性的角度，相信大部分家長是希望孩子最終能達到自由閲讀。然而要達到自由閲讀這個目標，可不是一蹴而就的，必須要有個五年的規劃。也就是我之前説的，至少要着眼五年，更長遠的是看十年後，期望孩子能達到和維持什麼樣的中文水平？

在此，我們特別強調，家長不要一開始就急於看到表現，千萬不能逐末捨本。學習中文和中國文化，不能像吃快餐那樣，不可能期望一年或幾個月就過中文關，認字關，這個不實際，也不符合語言文化學習的規律。

二、每個階段的目標

如果我們是着眼一個長期的目標，那麼進一步就是細分到每一個階段了。在這，我想着重提醒的是，每個階段發展的側重點有所不同，但並不是一刀切的，而是一個動態規劃。

繼續拿植物來做比喻，它在紮根的同時，也在長莖和葉。到了開花結果的時候，植物的根還是繼續往深度和廣度生長，它的莖和葉子也還是繼續繁茂的。

學習中文也一樣，最開始是打文化基礎，同時也在訓練聽説。到了一定程度，開始引入識字，但是文化學習和聽説訓練是仍然需要保持的。待到孩子的聽説能力培養起來，中文的理解能力培養起來，識字量一上去，閲讀就是水到渠成。在這個基礎上，寫作也可以逐步培養起來。有了這個大局觀，後面的規劃就好理解了。

總結一下，在孩子小的時候，把語言文化的根基打深打廣，孩子才有足夠的儲備，到了後面認字，閲讀和寫作，事半功倍，而且是四兩撥千斤，讓中文學習的路越走越寬廣。

同於自性，頤養天真。有教無類，樂學致遠。
