正心誠意，放聲朗讀

説起怎樣誦讀經典，我總結了八個字：“正心誠意，放聲朗讀。”

正心誠意，關鍵是帶讀的人要做到正心誠意，並把整個氣氛帶動起來。在家裏讀經，關鍵是家長。正如我們前面提到的“感知式”學習，家長是否認同讀經，是否誠心誠意，孩子都能感覺到的。可以説，我們到現在還沒有看到一個家庭，家長顧着上網打遊戲看八卦新聞，孩子能自動把經典讀好的。相反，我們看到的很多例子是，家長親自讀經，孩子自然而然接受讀經的。

放聲朗讀是指大家一起，做到“眼晴看着，心裏想着，口裏念着，耳朵聽着，手指指着。”你看，這樣，好多器官和功能都用上了，就不太容易走神，信息在腦子裏的作用也更大一些。

做到這八個字的目的，就是讓家長和孩子進入狀態。以前有的私塾老師讀經時，情不自禁地搖頭晃腦，就是進入了狀態。我並不是説大家要刻意搖頭晃腦地讀經，但是要體會那種專心致志的狀態。

2015年，很多家長説來觀摩我們的讀經小組活動。很多人來之前也許在預想着我們有什麼小竅門，小遊戲來帶動孩子讀經。但是她們來後，都無一例外地發現，我們的整個讀經過程非常簡單，就是家長和孩子圍坐一起，家長説：“孩子們，我們開始讀經啦！”然後，大家就開口朗讀。

為什麼我們會用這麼簡單的方式來進行讀經呢？原因有三個。

第一，建立一個好的氣氛。這個很重要，想想以前在國內，大家都想上重點中學，圖的就是學習的氣氛。大家在讀大學的時候，晚自習都喜歡去圖書館或者課室，一般不會在宿舍，也是圖個氣氛。

第二，當孩子坐下來，一起讀經的時候，往往那是他們注意力最好的時候。如果這個時候，我突然跳出來，跟孩子講個笑話，孩子們的心就飛走了，要把心再抓回來，定下來讀經，還真不容易。

第三，有助於培養良好的讀經習慣。

同於自性，頤養天真。有教無類，樂學致遠。
