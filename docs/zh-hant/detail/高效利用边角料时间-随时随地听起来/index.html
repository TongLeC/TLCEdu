高效利用邊角料時間，隨時隨地“聽”起來

古人學習一般是通過兩個途徑，一是拜師學藝，如果有幸跟着名師，每天老師口傳心授，進步可以很快。二是讀書，如果沒遇着名師，主要靠讀書了。俗話説“書中自有黃金屋，書中自有顏如玉。”靠讀書學習，就必須先過認字閲讀關，這也是近代語文教育都由認字閲讀入手的由來，因為文字是古時記錄知識的唯一媒體。

然而現在的科技發達了很多，文化知識還被大量記錄在音頻、視頻中。大家通過音頻、視頻也可以向名師學習，不一定需要先過認字閲讀關。換句話説，將來的孩子即使不懂漢字，只要能聽懂中文，也一樣可以瞭解中國文化的方方面面。

“聽”還是一個高效利用時間的方便法門。譬如我曾經苦惱自己找不到時間學習，因為每天忙完工作、孩子、家務，加上晚上讀個經，就基本找不出時間來學點新的東西了。後來我先生把《百家講壇》的音頻下載到一個MP3機，那樣我在開車的時候或者做家務的時候，就可以把一個講座聽完了。就好比是一位名師在我幹活的時候，陪伴着我，向我娓娓道來他的心得。

同樣道理，現在的孩子，學的東西廣泛得多了。除了學校的功課，課外英文閲讀，還有各種各樣的課後班，所以孩子們能拿來進行中文閲讀的時間就非常有限。如果他們能在坐車的時候，在家用餐的時候，早上起牀梳洗的時候，晚上洗澡的時候，又或者在家裏和兄弟姐妹玩耍的時候，順便聽上一個評書或講座，既立體使用時間，又在輕鬆的狀態下學習點知識和道理，何樂而不為呢？

家長只要各顯神通，邊角料時間就找出來了。

同於自性，頤養天真。有教無類，樂學致遠。
