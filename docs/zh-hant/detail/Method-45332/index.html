評書帶給孩子超乎想象的驚喜

大家都知道了聽評書的重要性，可以説是用最便捷的方式，收穫最全面的知識。根據我們多年的一線教學經驗，堅持聽評書的孩子，學習中文都會更輕鬆，而且對中文有內在興趣。聽評書究竟能給孩子帶來什麼呢？接下來，我們詳細論述：

豐富的語言

首先，評書裏有各種各樣的人物描寫，有老的，少的，男的，女的。女的也分肥的，瘦的，醜的，美的。美的女人，也分開不同的類型，有不同的描寫。四大美人，沉魚落雁，閉月羞花，各有各的典故，各有各不同的描寫語言。評書裏有大大小小各種不同的場景，既有千軍萬馬的戰場，又有恬靜的平湖秋月；既有高山流水，又有大海、平原和荒漠；既有宮殿高堂，又有民居草廬。每個場景描寫的語言都各不相同。評書裏還有錯綜複雜的故事情節，有各著名作家的優美措辭和豐富思想，有大量的成語，諺語，歇後語，詩詞，名句等等。想想我們平時跟孩子的中文交流，一般會是什麼呢？無非是吃喝拉睡，加上玩（去公園玩，好不好玩等）和學習（作業做了沒有？彈琴了沒有？等等）。我們的日常中文交流，跟評書的語言相比，是平淡單調得多了！所以説，海外孩子的中文語言訓練是非常缺乏的。而聽評書，正好可以幫助孩子補上語言聽力訓練。

中國傳統文化知識

評書裏的故事發生在各個不同的節日，例如春節，中秋，端午等，自然也會提到相應的節日風俗。評書的故事發生在中國幅員遼闊的大地，從嶺南到塞北，水鄉到大漠，內陸到邊疆，各地的風土人情在評書裏都自然會介紹到。評書的故事，還發生在不同的場合，對於各種宮廷禮節，婚嫁儀式，紅白事宜，等等，都會自然而然提到。而且在故事中提到的這些風俗、人情、禮節、儀式，是和生動的故事情節結合在一起的，孩子們更容易理解，更容易記住。這比我們單獨找個時間，把一個一個的風土人情，禮節儀式進行介紹要生動得多了。

大量的歷史人物和故事

一般的評書，是講述某一段歷史場景下錯綜複雜的故事。如《東周列國志》《楚漢爭霸》《三國演義》《興唐傳》《明朝那些事兒》等。故事裏有明君，也有昏君；有忠臣良將，也有奸臣小人。孩子們會聽到國家的分分合合，也聽到歷史朝代的變更。通過聽評書，孩子們對整個中國的歷史發展，基本就有一個概貌了。許多大家熟悉的歷史名人和典故，在評書裏都會講到。想一想，這麼多的歷史人物和典故，如果我每個晚上都給孩子們讀書，一個一個地講，要講多久啊？無論孩子們還是爸爸媽媽們都沒有這麼多時間來讀這些歷史故事。況且我們自己講也不一定能講得比語言藝術家生動啊!在聽了多部評書後，有一天，孩子就問我，能不能告訴他中國每個朝代的順序？於是我就把《朝代歌》找出來，給他解釋了一遍，孩子就很輕鬆地記住了大體的朝代順序。所以説，評書裏大量的歷史人物和故事，給孩子們奠定了很好的文化基礎。在後來的中文閲讀中，遇到的一些典故、名人，孩子們都能順利理解作者的意思。

學習歷史的好處很多，不同的人有不同的想法。正如《舊唐書·魏徵傳》所説：“以銅為鏡，可以正衣冠，以史為鏡，可以知興替，以人為鏡，可以明得失。”我讓孩子瞭解歷史故事，既是“以史為鏡”，也是“以人為鏡”。讓他們從歷史故事中學習做人的道理，理解社會和事物發展的規律，包括一些重要的哲學思想。譬如孩子們聽到王陽明“格竹子”的故事時，都不禁笑起來。後來聽到他統軍征戰，創下了很多神話，孩子們對他很多出奇制勝的謀略佩服得五體投地。我就順勢提到：“王陽明提倡‘知行合一’，就是‘要知，更要行，知中有行，行中有知’。我們在日常生活也該時時反觀自己，是否做到呀？”

正確的價值觀

這點也是我認為最重要的。因為評書裏有真善美，也有假惡醜；裏面提到了很多如何“修身、齊家、治國”的故事；也提到了什麼是“孝”，什麼是“仁、義、禮、智、信”。評書裏還有大量的故事講到中國人千百年來頌揚和傳承的價值觀與美德，如“勤勞”、“勇敢”、“節儉”等等。如果我們平時是苦口婆心地跟孩子説：“孩子，你要勤勞啊，你要勇敢啊，你要節儉啊。”孩子們會理解嗎，能接受嗎？枯燥的説教，其效果可能是孩子的煩躁；但是評書裏有活生生的人物、活生生的例子，加上生動的講述，孩子們潛移默化地就理解了這些價值觀。在聽故事的同時，適當地和孩子討論一下，給予引導，孩子們的大腦裏就會慢慢將經典裏的理論和評書中的實例結合起來了。

同於自性，頤養天真。有教無類，樂學致遠。
