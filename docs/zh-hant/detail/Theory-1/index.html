什麼是經典？

經典就是一個民族最精華最有價值的著作！經典出現在什麼時候？大概在公元前800年至公元前200年之間，尤其是公元前600年至前300年間，是人類文明的“軸心時代”。在軸心時代裏，各個文明都出現了偉大的精神導師，例如古希臘有蘇格拉底、柏拉圖、亞里士多德，以色列有猶太教的先知們，古印度有釋迦牟尼，中國有孔子、老子等等。他們提出的思想塑造了不同的文化傳統，也一直影響着人類的生活。

回到中國來，讓我們看看這文化棵樹。中國的經典就是中國的根文化，在後來的文化發展中，無論是唐詩宋詞元曲，還是明清小説，及近現代的小説，無論是四大名著還是其他重要的文章，都會看到經典的影子。

例如，《三字經》開篇是“人之初，性本善；性相近，習相遠。”這裏是引用了《論語∙陽貨》的“性相近也，習相遠也。”又例如《中華字經》裏有“春生夏長，秋收冬藏，時令應候，寒來暑往。”其實這裏就引用了《千字文》的第三句“寒來暑往，秋收冬藏。”大家可以看到，經典是得到廣泛的認同和引用的。

常常有家長會問，我們要讀哪些經典呢？一個原則是“什麼上乘，什麼有智慧，就讀什麼。”經典就是一個民族最精華最有價值的著作！不同的人，不同的民族對經典可能有不同的選擇。相同的是，這些經典都在千百年來指導着人們的思維模式和價值觀。

我們給同樂的孩子讀過的經典有：完整版的《道德經》、《大學》、《中庸》，節選版的《論語》、《孟莊荀列管墨名言》，《黃帝內經》、《詩經》、《易經》。不一定是最佳集合，僅供參考

其實，知識性強的歌訣也可以。例如，可以把現代的知識編成歌謠，先記憶，後理解。假如現在有哪位能人，把人體解剖學編成歌謠，我們也可以叫孩子先背誦，等將來再理解。道理是一樣的。
