從“根”上學中文，讓你意想不到的效果

打個比喻，一顆植物最吸引人，最能表現的部分就是它的花和果實。但是，如果沒有發達的根系，茂密的枝葉作為基礎，就不可能有長久的繁花碩果。甚至連植物本身的生存都有問題。在這裏，想分享一個真實的故事。

三年前，我家後院種了兩棵櫻桃樹。有一棵樹，第一年就結果了，但是長的葉子不多，也沒有長出多少新枝。另外一棵樹，第一年是一個果子都不結，枝葉卻長得很好，一年下來覆蓋了很大的範圍。在這問一下，大家會喜歡哪一棵樹呢？沒錯，有的朋友可能會選擇那棵結果子的樹。其實我開始也覺得第一年結果子的樹品種好。到了第二年，那棵在第一年就結果子的樹還是結了一些果子，但是隻有十幾個，而且葉子還是不多。我開始隱隱擔憂了。那棵第一年不結果的樹，在第二年只結了一個，它還是拼命長葉子。到了第三年，大家猜猜情況如何？那棵在前兩年努力結果子的樹死了，而那棵拼命長葉子的樹，還活得好好的，而且結了二十多個果。

當時，我看那棵枯死掉的果樹所在地方周圍空間更大一些，心想不如把那棵活得好好的樹移植過去。挖開泥土的時候，我發現第一年就結果子的櫻桃樹，根系還是小小的，但是那棵一直拼命長葉子的櫻桃樹，其根部非常發達。這個時候，大家再想想你會喜歡哪一棵櫻桃樹？相信心裏都有答案了。

通過上面的例子，我們可以看到，中文學習中的“文化、聽、説、讀、寫”各部分的關係，和植物的“根、莖、葉、花、果”的關係有點相似。

讓我們來看一下上面這幅圖。讓孩子打好中文的文化基礎，猶如植物把根扎深扎廣；進行大量的語言“聽説”訓練，好比是植物把莖和葉子（也就是軀幹）長好；而“讀寫”，好比是花果，是大多數家長急切想看到的結果。

從一棵植物生長的角度來説，是按照“根莖葉花果”次序，即先紮根，再抽葉，長莖，最後開花結果。站在中文學習的角度，從長遠的整體規劃來看，也必須遵循這個次序：先紮下文化的根，訓練聽説，再到認字，最後達到讀寫。順應這個基本規律與次序，才有可能得到可持續的發展，並且事半功倍。在這裏，我們講的次序是指各個時期學習側重點的次序，實際上各項學習內容在不同時期都是並行，而且相互關聯的。

同於自性，頤養天真。有教無類，樂學致遠。
