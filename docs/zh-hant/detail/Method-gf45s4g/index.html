拒絕啞巴中文，鼓勵孩子放聲誦讀

“説”是語言的表達和發放能力。有時候知識和思想在腦子裏，不一定能自如説出來。如果訓練不夠，孩子長大後説中文會結結巴巴，就不願意用中文去交流的。我們在國內學習英文時就往往有這個問題，英文書看得懂，但到了嘴邊，卻像“茶壺裏煮餃子，肚裏有貨倒不出”，這就是練習不足所致。

為什麼“説”的訓練那麼重要呢？這首先要看看人類是如何完成語言的發音的。

上面的圖畫，介紹了一部分發音用到的器官，其實還有很多，譬如肺，胸肌，腹肌等等。除了器官，還需要大腦裏聯繫知識和支配運動的神經系統的配合。可以説，一個簡單的發音，就等於動用了“多支部隊的協同作戰”才完成的，沒有訓練，哪裏來的協調呢？況且每一種語言的發音規律是不同的。即使是中國國內不同地方的方言，音調也有所不同。例如普通話有4個聲調，上海話有5-6個聲調，粵語則有9-10個聲調，每一個聲調的震動方式是不同的，都是需要練習的。我們時常會聽到外地人學習本地方言時發出的怪腔怪調，就是因為有些發音部位沒有經過充分訓練的結果。這就譬如打慣少林拳的，打太極拳就沒有那麼自如，還需要另外的訓練。

誦讀的第一個最直接的好處，就是鍛鍊了語言的發放能力，增強了語感。

同於自性，頤養天真。有教無類，樂學致遠。
