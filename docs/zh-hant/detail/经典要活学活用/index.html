經典要活學活用

前面提到在經典中，我們可以向古聖先賢學習其智慧和精神。那麼怎麼做到活學活用呢？建議從下面幾個方面入手。

一、切忌紙上談兵

我們都知道“紙上談兵”這個故事。趙括熟讀兵書，缺乏實戰經驗；卻自以為是。結果導致了四十萬趙軍被殺的慘痛悲劇。長平一戰之後，趙國再無力對抗秦國，直接促成了趙國亡國。以至後代的史學家感慨道“死讀兵書，不如不讀兵書。”同樣的，我們學習經典，切忌生搬硬套。

有個誤區，就是認為只要讀了四書五經，就能成大材。其實讀經只是一個起步，只是培養人材的一塊重要基石。大家可以看看，在晚清時期，朝廷中就不乏通讀四書五經的文人，但是能領悟其中精髓，並結合實際情況去實踐的人並不多，相反，不少人是死讀經典而不開化的腐儒。當時有這麼一段耐人尋味的故事。據説李鴻章從英國海軍司令何伯處得了兩個千里鏡，也就是單筒望遠鏡，他趕緊送了一個給恩師兼上司曾國藩。當時曾國藩一看，驚訝不已，他不但心裏不停琢磨其中的原理，而且感悟地説：“天下之物，凡加倍磨冶，皆可變換本質，別生精彩”同時他馬上命令手下人向何伯買來幾十個千里鏡，為炮兵打仗時提供方便。然而，當時在旁邊的一些大臣，卻是抱着不屑的態度，認為這是洋人的奇巧之器，我清朝泱泱大國，根本不需要這玩意。試想一下，如果當時清政府上下都是這樣自大自封的書蟲佔據着，如果當時沒有曾國藩，李鴻章這樣既熟讀經典，又領悟其中道理，更能實事求是地實踐的忠臣來指引朝政方向的話，就不會有中國近代的洋務運動。

經典的智慧，如果僅僅是以文字的形式存在腦子裏，可以説是沒有任何價值的。但是如果自恃肚子裏有幾句經典的墨水，而驕傲自大，拒絕接觸新的事物和新思想，甚至以此嘲笑別人的虛心學習，那就太可怕了！所以我們學習經典，不能只留於紙上，不能僅滿足於出口成誦。

二、直指本質

學習經典，不能只學字面意思，還要看文字背後的本質思想。

在《論語·雍也》裏有這一段。宰我問曰：“仁者，雖告之曰：‘井有仁焉。’其從之也？” 子曰：“何為其然也？君子可逝也，不可陷也；可欺也，不可罔也。”意思是宰我問孔子：“追求仁德的人，假如別人告訴他井裏掉下去一位仁人啦，他應該跟着下去嗎？”言下之意是跳下去救，即使死了也是“仁”和“見義勇為”的表現，但不救就是“見死不救”和“不仁”。然而孔子説：“為什麼要這樣做呢？君子可以到井邊去救，卻不可以陷入井中；君子可能被欺騙，但不可能被迷惑。”由此可見，孔子更注重於救人的本質，也就是看最後結果，而不是行為的表面性。如果是徒然危害自己，同時對別人沒有好處的行為，是不可取的。

這還讓我們聯想起尾生的故事。春秋時，魯國曲阜有個年輕人名叫尾生，與聖人孔子是同鄉。尾生為人正直，樂於助人，和朋友交往很守信用，受到四鄉八鄰的普遍讚譽。有一次，他的一位親戚家裏醋用完了，來向尾生借，恰好尾生家也沒有醋，但他並不回絕，便説：“你稍等一下，我裏屋還有，這就進去拿來。”尾生悄悄從後門出去，立即向鄰居借了一罈醋，並説這是自己的，就送給了那位親戚。孔子知道這件事後，批評尾生為人不誠實，有點弄虛作假。尾生卻不以為然，他認為幫助別人是應該的，雖然説了謊，但出發點是對的。對於這件事，我們不得不佩服孔子，他提倡助人為樂，但反對做表面文章的善行。所謂“有心為善即為惡”，指的是為了強求做善事，即使時機不成熟也強求着去做，其實是為了粉飾出一個善的表面，而忽略了善的本質。真正的善是時時保持善念，遇到需要幫忙的，隨手就做，不論大小，也不矯揉造作。

三、知行合一

知行合一，是王陽明心學的核心理念之一。他的一生把“知行合一”演繹得淋漓盡致。

晚清四大名臣之一的曾國藩也堪稱典範。其修身，治家，為官之道得到人們的廣泛稱讚。他在寫給諸弟的一封家書中談到：“絕大學問即在家庭日用之間”。他認為：“讀聖賢之書在於領悟其中的道理，領悟道理在於指導生活實踐，只領悟其中道理，而不身體力行地去實踐，不僅無益，反而有害。”

説到這，不得不提一下尾生的另一段故事。《莊子·盜蹠》：“尾生與女子期於梁(橋)下，女子不來，水至不去，抱樑柱而死。”這是講他在梁地認識一位漂亮姑娘。兩人一見鍾情，私訂終身。但姑娘的父母嫌棄尾生家境貧寒，堅決反親事。為了追求愛情和幸福，姑娘決定揹着父母私奔。那一天，兩人約定在韓城外的一座木橋邊會面，雙雙遠走高飛。黃昏時分，尾生提前來到橋上。不料，六月的天説變就變，突然狂風怒吼，雷鳴電閃，滂沱大雨傾盆而下。緊接山洪暴發，滾滾江水席捲而來，淹沒了橋面，沒過了尾生的膝蓋。“城外橋面，不見不散！”尾生想起了與姑娘的誓言，四顧茫茫水世界，不見姑娘蹤影，仍寸步不離，死抱着橋柱，終被活活淹死。再説姑娘因為私奔念頭泄露，被父母禁錮家中，不得脱身。她午夜後伺機逃出家門，冒雨來到城外橋邊。此時洪水已退，姑娘看到緊抱橋柱而死的尾生，悲慟欲絕。姑娘哭罷，便相擁縱身投入江中，譜寫一幕驚心動魄的愛情悲劇。尾生信守諾言是美好的，但卻被僵化地執行，不懂得隨着情況的變化而變通，其結果也是可悲的。其實尾生完全可以爬上一棵樹或跑到高地等候。只要留得青山在，不怕沒柴燒，若是青山不再，生命不復，什麼諾言都無法實踐。

我們學習經典，關鍵是領悟其精髓，然後處於什麼情況，就適當地調整，靈活應用。

四、與時俱進

社會是向前發展的。不僅科技在進步，而且各種禮儀和觀念也在演變和發展中。如果不與時俱進，只是墨守成規，最後也只能是與時代格格不入。

在《論語·八佾》裏提到：子貢欲去告朔之餼羊。子曰：“賜也！爾愛其羊，我愛其禮。”意思是説子貢提出去掉每月初一舉行告朔之禮所供奉的腥羊。孔子説：“賜，你愛惜那隻羊，我卻愛惜那種禮。”孔子時代對禮是非常尊崇的，並且嚴格執行，所以在《論語》裏多次討論到禮。但是隨着時代的變遷，禮也在變化。我們不可能完全按照古人的方式去執行禮。例如古人用豬羊祭祖，今人可以改為敬鮮花果品。方式變了，但是“禮”背後的實質是一樣的，就是創造一種氛圍，讓人們心裏生起恭敬心。時時保持對萬物的恭敬心本身既是一種方法，也是一種境界。

總結一下，學習經典，並不是死記硬背，而是從中學習古人的智慧和好的精神。經典中的道理都是活潑生動，不僵化死板的；我們需要的是洞察這些做人道理背後的本質，融匯貫通，不停地思考，不斷地進步，並加以靈活運用。

同於自性，頤養天真。有教無類，樂學致遠。
